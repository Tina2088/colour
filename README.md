# ChromaVision: Art Student Color Sensitivity Challenge

A precision color sensitivity game designed for art students to test and improve their chromatic perception.

## Features
- **Bilingual Support**: Toggle between English and Chinese.
- **Adaptive Difficulty**: Color differences decrease as you level up.
- **Performance Analysis**: Get a detailed report on your color sensitivity.
- **Responsive Design**: Play on mobile or desktop.

## Tech Stack
- React 19
- Vite
- Tailwind CSS 4
- Framer Motion
- Lucide Icons

## Deployment

### GitHub
1. Create a new repository on GitHub.
2. Initialize your local directory as a git repo:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   ```
3. Link to your GitHub repo and push:
   ```bash
   git remote add origin <your-repo-url>
   git branch -M main
   git push -u origin main
   ```

### Vercel
1. Go to [Vercel](https://vercel.com).
2. Click "Add New" -> "Project".
3. Import your GitHub repository.
4. Vercel will automatically detect Vite. Click "Deploy".
5. Your app will be live in seconds!
